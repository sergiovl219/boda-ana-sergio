## Optimización

Para el mejor rendimiento en la invitación web:
- Usa formato **WebP** o **JPEG** optimizado
- Comprime las imágenes con [TinyPNG](https://tinypng.com/) o [Squoosh](https://squoosh.app/)
- La foto del hero (`hero-bg.jpg`) es la más importante — usa la máxima calidad posible
- Las fotos del timeline pueden comprimirse más agresivamente (calidad ~80%)
